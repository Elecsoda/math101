1 · 周长是“一圈有多长”，面积是“占了多少格”

先不用任何公式。周长就是沿着图形边线走一圈的总长度——从起点出发、顺时针一段一段按顺序数线段，不重不漏；面积就是图形盖住了多少个方格——一个一个地数格子。下面在网格里画一个图形，点按钮亲手数一数。

边线（数线段 → 周长） 方格（数格子 → 面积） 每个小格 = 边长 1 的正方形
点上面的按钮开始数
周长 = 走过 0 段 = 0 面积 = 盖住 0 格 = 0
周长沿着边线数，单位是长度（如厘米）；面积数里面的方格，单位是面积（如平方厘米）。它们数的东西不一样——周长数“边”，面积数“面”。

2 · 图形的周长 = 把每条边加起来（相同的边用乘法更快）

周长就是所有边长之和。先把每条边一条一条加起来；当图形有相同长度的边时，与其反复相加，不如用乘法简化（如正方形 边长×4、正多边形 边长×边数）——下面会把乘法捷径高亮出来。

3 · 面积起步：从正方形到长方形，先数格子

面积就是数里面有多少个单位方格。先看正方形：每行同样多，有几行就是几个那么多。再看长方形：一行有 长 个方格，一共有 宽 行——不用一个个数，每行个数 × 行数 就能算出来。这就是“面积 = 长 × 宽”的来历。

4 · 割补法：把新图形“变”成会算的图形

不认识的图形怎么求面积？剪一刀平移或复制一个旋转拼上，变成已经会算的图形，面积不变就能算了。平行四边形沿高剪下三角形平移→长方形；三角形、梯形则复制一个绕中心旋转 180°拼上→平行四边形，原图是它的一半。点“割补演示”看过程。
